エグゼクティブサマリー

Google I/O 2026（2026年5月19-20日開催）の全体テーマは「AIエージェント時代の本格到来」。Gemini 3.5 Flash / Pro の新モデル発表を軸に、24時間稼働のパーソナルエージェント Gemini Spark、エージェント開発基盤 Antigravity 2.0、25年以上ぶりの検索ボックス刷新、Google横断のユニバーサルカートなど、100以上の発表が行われた。

AI Modeは月間10億ユーザーを突破し、Personal Intelligenceは200カ国・98言語に拡大。開発者向けには Managed Agents API、Migration Agent（他フレームワーク→Kotlin自動変換）、WebMCP など、エージェント前提の新ツールが多数発表された。

サブスクリプション体系もコンピュートベース課金へ移行し、新たに Google AI Ultra（$100/月）が追加。ハードウェアでは Samsung/Gentle Monster/Warby Parker との共同開発による Android XR スマートグラスが2026年秋に登場予定。

コードサンプル: Managed Agents API

Managed Agents を使って単一のAPI呼び出しでエージェント環境を構築する例（Python）:

import google.generativeai as genai

# Managed Agent の作成
client = genai.Client(api_key="YOUR_API_KEY")

agent = client.agents.create(
    model="gemini-3.5-flash",
    instructions="""
    You are a code review agent.
    Analyze the given repository and provide
    a detailed security audit report.
    """,
    tools=["code_execution", "web_browse"],
    sandbox=True
)

# エージェントにタスクを送信
result = agent.run(
    prompt="Review the authentication module for vulnerabilities",
    files=["./src/auth/"]
)

print(result.report)

※ 上記はGoogle公式ブログの説明に基づくコンセプト例です。正式なAPIドキュメントは ai.google.dev を参照してください。

開発者への影響

中長期的な戦略変更

  • エージェントファースト設計: Antigravity 2.0 / WebMCP の登場により、アプリ設計はエージェントとの連携を前提にすべき時代に
  • Migration Agent の活用検討: React Native / iOS プロジェクトの Kotlin ネイティブ移行コストが大幅低減

  • コンピュートベース課金への対応: API 利用コストの予測モデルを見直す必要あり
